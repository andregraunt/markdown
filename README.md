которые вычисляются по формуле

`x = (-b +- sqrt(b^2-4ac))/(2a) .`

Если \(a \ne 0\), то существуют 
два решения уравнения \(ax^2 + bx + c = 0\), 
которые вычисляются по формуле
$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
